Tutorial


Canvas field

the Canvas field is defined by border (grey-rectangle). it has several events, as listed below:

  • Mouse Event : will occur when your cursor is in canvas field.
    • Mouse-Wheel : easily zoom-in and zoom-out using your mouse wheel.
    • Hold left-click : move around the canvas to change the view, work similarly like scroll-bar.
    • single right-click : zoom-in
    • right-click + Shift-Key : zoom-out

About

The program allows user to create Polygon mesh in web platform using canvas API. The polygons are represented using a list of vertex coordinates.
